---------------------------------О "PANZERCHASM"--------------------------------

"PanzerChasm" - свободное воссоздание игры "Chash - The Rift".
Для работы требуются данные оригинальной игры (файл CSM.BIN).


------------------------------СИСТЕМНЫЕ ТРЕБОВАНИЯ------------------------------

* Клавиатура, мышь.
* Операционная система Windows XP или более новая.
* Поддержка OpenGL 3.3 в системе/драйверах.


----------------------------ИНСТРУКЦИЯ ПО ПРИМЕНЕНИЮ----------------------------

Поместить CSM.BIN от оригинальной игры в директорию с PanzerChasm.exe. Запустить
PanzerChasm.exe.

Управление: WASD - передвижение, SPACE - прыжок, левая кнопка мыши - стрелять,
цифры 1-8 - выбор оружия, "~" - открыть/закрыть консоль, ESC - меню.

В консоли есть ряд команд. Чтобы увидеть их список, нунжо нажать TAB.
